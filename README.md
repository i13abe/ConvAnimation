# How to use
## Install poetry
Python のpackage管理用にPoetryを利用する.
see: https://python-poetry.org/docs/#installation

## Install libraries
poetryでパッケージをインストールする.
```sh
$ poetry install
```

pytorchのみ自前で取ってくる.
以下のリンクより適した環境のものを取得.
https://pytorch.org/

## Run and get animation
```sh
$ make anim
```
